## ✅ Flujo de ejecución

1. Levantar **PostgreSQL**  
2. Instala las dependencias del proyecto con tu manejador preferido
```bash
# Usando pnpm
pnpm install

# Alternativa con npm
npm install

# Alternativa con yarn
yarn install
```
3. Iniciar el **backend** con tu manejador preferido:
```bash
# Usando pnpm
pnpm dev

# Alternativa con npm
npm run dev

# Alternativa con yarn
yarn dev
``` 
4. Servidor disponible en:  
👉 http://localhost:4000

---

## 📌 Notas
- Requiere **Node.js ≥ 18** y **PostgreSQL ≥ 14**  
- Mantén tu archivo `.env` fuera del repositorio para mayor seguridad
